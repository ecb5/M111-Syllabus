    Section 10.11 Concavity

    Exercises Exercises

    Exercise Group.

    Identify all intervals where the given curve is concave up, and where it is concave down. Check your work using the Desmos interface below after you have solved each problem.
    1.
    \(\displaystyle y=x^3-2x^2+x+1\)
    Solution.
    Concave up: \((2/3,\infty)\)
    Concave down: \((-\infty, 2/3)\)
    2.
    \(\displaystyle y= 3x^4-4x^3+6\)
    Solution.
    Concave up: \((-\infty, 0)\cup(2/3,\infty)\)
    Concave down: \((0,2/3)\)
    3.
    \(\displaystyle y=2x^6-6x^4\)
    Solution.
    Concave up: \((-\infty, -\sqrt{6/5})\cup(\sqrt{6/5},\infty)\)
    Concave down: \((-\sqrt{6/5},\sqrt{6/5}\)
    4.
    \(\displaystyle y=(x^2-1)^2\)
    Solution.
    Concave up: \((-\infty, -1/\sqrt{3})\cup(1/\sqrt{3}, \infty)\)
    Concave down: \((-1/\sqrt{3},1/\sqrt{3})\)
    5.
    \(\displaystyle y=\sqrt[5]{x}-1\)
    Solution.
    Concave up: \((-\infty, 0)\)
    Concave down: \((0, \infty)\)
    6.
    \(\displaystyle y=x^2-\frac{27}{x^2}\)
    Solution.
    Concave up: \((-\infty, -3)\cup(3,\infty) \)
    Concave down: \((-3,3)\)
    7.
    \(\displaystyle y=\frac{x}{x^2+1}\)
    Solution.
    Concave up: \((-\sqrt{3},0)\cup(\sqrt{3},\infty)\)
    Concave down: \((-\infty, -\sqrt{3})\cup(0,\sqrt{3})\)
    8.
    \(\displaystyle y=\sqrt[3]{x^2}(3x+10)\)
    Solution.
    Concave up: \((2/3, \infty)\)
    Concave down: \((-\infty, 0)\cup(0, 2/3)\)
    9.
    \(\displaystyle y=8x^{\frac13}+x^{\frac43}\)
    Solution.
    Concave up: \((-\infty,0)\cup(4,\infty)\)
    Concave down: \((0,4)\)

    Exercise Group.

    For each of the following sketch the graph of a continuous function, \(y=f(x)\) which satisfies all of the stated conditions.
    10.
    \begin{align*} f(0)=1;f(2)=3\\ f^\prime(0)=f^\prime(2)=0\\ f^\prime(x)\lt 0 \text{ if } \abs{x-1}\gt 1\\ f^\prime(x)\gt 0 \text{ if } \abs{x-1}\lt 1\\ f^{\prime\prime}(x)\gt 0 \text{ if } x\lt 1\\ f^{\prime\prime}(x)\lt 0 \text{ if } x\gt 1 \end{align*}
    11.
    \begin{align*} f(0)=4; f(2)=2; f(5)=6\\ f^\prime(0)=f^\prime(2)=0\\ f^\prime(x)\gt 0 \text{ if } \abs{x-1}\gt 1\\ f^\prime(x)\lt 0 \text{ if } \abs{x-1}\lt 1\\ f^{\prime\prime}(x)\lt 0 \text{ if } x\lt 0 \text{ or if } \abs{x-4}\lt 1 \text{ or if } x\gt 5 \end{align*}
    12.
    \begin{align*} f(0)=2; f(2)=f(-2)=1\\ f^\prime(0)=0\\ f^\prime(x)\gt0 \text{ if } x\lt0\\ f^\prime(x)\lt0 \text{ if } x\gt0\\ f^{\prime\prime}(x)\lt0 \text{ if }\abs{x}\lt2\\ f^{\prime\prime}(x)\gt0 \text{ if }\abs{x}\gt2 \end{align*}
    13.
    \begin{align*} f(1)=4\\ f^\prime(x)\gt 0 \text{ if } x\lt1\\ f^\prime(x)\lt 0 \text{ if } x\gt1\\ f^{\prime\prime}(x)\gt0 \text{ for all } x\neq1 \end{align*}
