    Section 10.17 Integration by Substitution

    Exercises Elementary Substitutions

    Indefinite Integrals.

    Compute each of the following indefinite integrals (antiderivatives).
    1.
    \(\displaystyle\int 3x^2(x^3-1)^5\dx{x}\)
    Solution.
    \(\frac{(x^3-1)^6}{6}+C\)
    2.
    \(\displaystyle\int 3x^2(x^3-1)^5\dx{x}\)
    Solution.
    \(\frac{(x^3-1)^6}{6}+C\)
    3.
    \(\displaystyle\int \sin(x)\cos(x)\dx{x}\)
    Solution.
    \(\frac{\sin^2(x)}{2}+C = -\frac{\cos^2(x)}{2}+C\)
    4.
    \(\displaystyle\int \frac{\sin\sqrt{x}}{\sqrt{x}}\dx{x}\)
    Solution.
    \(-2\cos\left(\sqrt{x}\right)+C\)
    5.
    \(\displaystyle \int \frac{1}{5-3x} \dx{x}\)
    Solution.
    \(-\frac13\ln\abs{5-3x})+C\)
    6.
    \(\displaystyle\int \frac{4}{(1+2x)^3}\dx{x}\)
    Solution.
    \(\frac{-1}{(1+2x)^2}+C\)
    7.
    \(\displaystyle\int \sin^2(x)\cos(x)\dx{x}\)
    Solution.
    \(\frac{\sin^3(x)}{3}+C\)
    8.
    \(\displaystyle\int \cos^2(x)\sin(x)\dx{x}\)
    Solution.
    \(-\frac{\cos^3(x)}{3}+C\)
    9.
    \(\displaystyle\int \cos^3(x)\sin^3(x)\dx{x}\)
    Solution.
    \(\frac{\sin^4(x)}{4}-\frac{\sin^6(x)}{6}+C\)
    10.
    \(\displaystyle\int \frac{e^x}{1+e^{2x}}\dx{x}\)
    Solution.
    \(\inverse\tan(e^x)+C\)
    11.
    \(\displaystyle\int \sqrt{4-2x}\dx{x}\)
    Solution.
    \(\frac{(4-2x)^{\frac32}}{3}+C\)
    12.
    \(\displaystyle\int \sqrt[3]{4-2x}\dx{x}\)
    Solution.
    \(\frac34(4-2x)^{\frac43}+C\)
    13.
    \(\displaystyle \int \frac{e^x}{1+e^x}\dx{x}\)
    Solution.
    \(\ln(1+e^x)+C\)
    14.
    \(\displaystyle\int 7x\sqrt[5]{x^2+2}\dx{x}\)
    Solution.
    \(\frac{35}{12}(x^2+2)^{\frac52}+C\)
    15.
    \(\displaystyle\int \sec^3(x)\tan(x)\dx{x}\)
    Solution.
    \(\frac{\sec^3(x)}{3}+C\)
    16.
    \(\displaystyle\int \frac{\sin(\pi/x)}{x^2}\dx{x}\)
    Solution.
    \(\frac{1}{\pi}\cos(\pi/x)+C\)
    17.
    \(\displaystyle\int x^5\sqrt[3]{x^3+2}\dx{x}\)
    Solution.
    \(\frac17(x^3+2)^{\frac73}-\frac32(x^3+2)^{\frac43}+C\)
    18.
    \(\displaystyle\int \frac{x}{\sqrt[4]{x+2}}\dx{x}\)
    Solution.
    \(\frac47(x+2)^{\frac74}-\frac83(x+2)^{\frac34}+C\)
    19.
    \(\displaystyle\int \frac{x^2}{\sqrt{1-x}}\dx{x}\)
    Solution.
    \(-\left(2(1-x)^{1/2}-\frac43(1-x)^{3/2}+\frac25(1-x)^{5/2}\right)+C\)
    20.
    \(\displaystyle\int \frac{1}{\sqrt{1+\sqrt{1+x}}}\dx{x}\)
    Solution.
    \(-\frac83\left(\sqrt{1+\sqrt{1+x}+4\sqrt{1+x+(1+x)^{\frac32}}}\right)+C\)
    21.
    \(\displaystyle\int \frac{\cos(x)\sin(x)}{\sqrt{\cos^2(x)+16}}\dx{x}\)
    Solution.
    \(-\sqrt{\cos^2(x)+16}+C\)
    22.
    \(\displaystyle\int x\sqrt{1-\sqrt{x}}\dx{x}\)
    Solution.
    \(\frac25(1-\sqrt{x})^{5/2}+C\)
    23.
    \(\displaystyle\int \sqrt{x-x\sqrt{x}}\dx{x}\)
    Solution.
    \(-(1-\sqrt{x})^{3/2}+C\)

    Definite Integrals.

    Compute each of the following definite integrals.
    24.
    \(\displaystyle \int_{0}^{1} \cos(\pi x)\dx{x}\)
    Solution.
    \(0\)
    25.
    \(\displaystyle \int_{0}^{\pi/4} \sin(4x)\dx{x}\)
    Solution.
    \(\frac12\)
    26.
    \(\displaystyle \int_{0}^{2} (x-1)^{25}\dx{x}\)
    Solution.
    \(0\)
    27.
    \(\displaystyle \int_{0}^{7} \sqrt{4+3x}\dx{x}\)
    Solution.
    \(26\)
    28.
    \(\displaystyle \int_{0}^{1} x^2(1+2x^3)^5\dx{x}\)
    Solution.
    \(\frac{182}{9}\)
    29.
    \(\displaystyle \int_{0}^{\sqrt{\pi}} x\cos(x^2)\dx{x}\)
    Solution.
    \(=\frac{\sin\left(13^2\right)}{2}\approx 0.301\)
    30.
    \(\displaystyle \int_{0}^{13} \frac{1}{\sqrt[3]{(1+2x)^2}} \dx{x}\)
    Solution.
    \(3\)
    31.
    \(\displaystyle \int_{-\pi/3}^{\pi/3} \sin^5(x)\dx{x}\)
    Solution.
    \(0\)
    32.
    \(\displaystyle \int_{0}^{4} \frac{x}{\sqrt{1+2x}}\dx{x}\)
    Solution.
    \(\frac{10}{3}\)
    33.
    \(\displaystyle \int_{0}^{3} \frac{1}{2x+3} \dx{x}\)
    Solution.
    \(\ln\left(\sqrt{3}\right)\)
    34.
    \(\displaystyle \int_{0}^{1/2} \frac{\inverse\sin(x)}{\sqrt{1-x^2}}\dx{x}\)
    Solution.
    \(\frac{\pi^2}{72}\)
    35.
    \(\displaystyle \int_{e}^{e^4} \frac{1}{x\ln(x)}\dx{x}\)
    Solution.
    \(\ln{4}\approx 1.38629\)

    36. Find The Pattern.

    (a)
    Show that \(\int_0^1x^2(1-x)^3\dx{x} = \int_0^1x^3(1-x)^2\dx{x}\)
    Hint.
    Try the substitution: \(u=1-x\text{.}\) Don't lose track of the negatives.
    (b)
    \(\int_0^1x^5(1-x)^6\dx{x} = \int_0^1x^6(1-x)^5\dx{x}\)
    (c)
    \(\int_0^1x^a(1-x)^3\dx{x} = \int_0^1x^3(1-x)^a\dx{x}\)
    (d)
    \(\int_0^1x^3(1-x)^b\dx{x} = \int_0^1x^b(1-x)^3\dx{x}\)
    (e)
    \(\int_0^1x^a(1-x)^b\dx{x} = \int_0^1x^b(1-x)^a\dx{x}\)
    (Assume that \(a\) and \(b\) are positive numbers.)
    (f)
    Use the pattern you found in the problems above to show that if \(f\) is integrable then
    \begin{gather*} \int_0^1f(x)\dx{x} = \int_0^1f(1-x)\dx{x} \end{gather*}

    37. Find The Pattern.

    Find a function, \(y=f(x)\text{,}\) which satisfies each of the following equations.
    (a)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^2\)
    Hint.
    Try differentiating.
    (b)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^3\)
    (c)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^4\)
    (d)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^n\)
    (e)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^{1/2}\)
    (f)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^{1/3}\)
    (g)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^{1/4}\)
    (h)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]^{1/n}\)
    (i)
    \(\displaystyle \int_0^xf(t)\dx{t} = [f(x)]\)

    38.

    Suppose \(f(x)\) is a continuous function in the interval \([0,1]\) and that \(\int_0^1f(x)\dx{x} = \frac13\text{.}\)
    Assume that each of the following is also true on the same interval.
    \begin{gather*} f(0)=0,\\ f(1)=1,\\ f^\prime(x)>0. \end{gather*}
    Show that \(\displaystyle\int_0^1\inverse{f}(y)\dx{y} = \frac23.\)
    Hint.
    There is nothing to compute here. Try drawing a picture of the problem.

    Find the Pattern:.

    Show that each of the following is true:
    39.
    \(\int_0^2\sin\sqrt{x}\dx{x} = 2\int_0^4x\sin(x)\dx{x} \)
    40.
    \(\int_0^2\sin\left(x^{1/3}\right)\dx{x} = 3\int_0^8x^{2}\sin(x)\dx{x}\)
    41.
    \(\int_0^2\sin\left(x^{1/4}\right)\dx{x} = 4\int_0^{16}x^{3}\sin(x)\dx{x}\)
    42.
    \(\int_0^2\sin\left(x^{1/n}\right)\dx{x} = n\int_0^{2^n} x^{n-1}\sin(x) \dx{x}\)

    43.

    Use the pattern you found in the previous problems to show that \(\int_0^2\sin\left(x^{m/n}\right)\dx{x}\) is equal to each of the following:
    (a)
    \(n\int_0^{2^{n}}x^{n-1}\sin(x^m)\dx{x}\)
    (b)
    \(\frac{n}{m}\int_0^{2^{nm}}x^{\frac{n}{m}-1}\sin(x) \dx{x}\)

    44. Generalize The Pattern.

    Show that
    \begin{gather*} \int_0^2f\left(x^{m/n}\right)\dx{x} \end{gather*}
    is equal to each of the following:
    (a)
    \(\displaystyle n\int_0^{2^{n}}x^{n-1}f(x^m)\dx{x}\)
    (b)
    \(\displaystyle\frac{n}{m}\int_0^{2^{nm}}x^{\frac{n}{m}-1}f(x) \dx{x}\)

    45.

    Use the pattern you found in the previous problems to find an integral that is equal to
    \begin{gather*} \displaystyle \int_a^bf\left(x^{m/n}\right)\dx{x} \end{gather*}
